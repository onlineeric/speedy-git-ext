### get avatar from GitHub analysis
Write a function or module to fetch a user's GitHub avatar without requiring any API authorization.

Implementation details:
- Endpoint: Use the public GitHub API, such as fetching commit details via /repos/{owner}/{repo}/commits/{commit_hash} or user details via /users/{username}, and extract the author.avatar_url from the JSON response.
- Action: Download the image from the avatar_url and cache it locally per account.

Limits to handle:
- Rate Limit: Unauthenticated GitHub API requests are strictly limited to 60 requests per hour per IP address.
- Header Monitoring: You must check the x-ratelimit-remaining header in the HTTP response. If it equals '0', pause all GitHub API requests until the timestamp specified in the x-ratelimit-reset header.

Notes & Fallback strategy:
- Fallback: If the GitHub API returns an error, the rate limit is exceeded, or no avatar is found, automatically fallback to using Gravatar.
