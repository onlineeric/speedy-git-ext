## Execution Steps

Execute the below steps one by one, one after another.

1. Run `/speckit.plan` to plan the feature.
2. Run `/speckit.tasks` to generate the tasks.
3. Run `/speckit.analyze` to analyze the tasks.
4. Fix all issues found in the analysis by your best suggestion.
5. Run `/speckit.implement` to implement the feature.
